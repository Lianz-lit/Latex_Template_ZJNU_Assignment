## 介绍
- 用于浙师大(ZJNU)研究生课程论文/期末报告的Latex模板

## 模板下载

* 已经上传至Overleaf模板，点击即用 ☞ [Overleaf](https://)

## 使用方法
默认模板文件由以下四部分组成：

- `main.tex` 主文件,包含报告正文
- `reference.bib` 参考文献列表
- `ZJNUReport.sty` 文档格式控制，包括一些基础的设置，可以设置页眉、标题、姓名等

使用时需在`ZJNUReport.sty` 对课程、姓名、学号、学院、页眉等进行设置，然后编辑`main.tex`编写正文
